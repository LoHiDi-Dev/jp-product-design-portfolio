# Joel Stefano Premier | Portfolio

A portfolio website showcasing UX/product design work, case studies, and proof of impact.

## Run locally

```bash
pnpm install
pnpm dev
```

Open http://127.0.0.1:5173 (or the port shown in the terminal).

## Build

```bash
pnpm build
pnpm start   # preview production build
```

## Stack

Vite, React, React Router, Tailwind CSS.
